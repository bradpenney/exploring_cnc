# Exploring CNC

CNC machining and CAD/CAM from first principles — coordinate systems, G-code, FreeCAD workflows, and open-source machine control. Part of the [BradPenney.io](https://bradpenney.io) learning ecosystem.

**Live Site:** [cnc.bradpenney.io](https://cnc.bradpenney.io)

## Development

```bash
# Install dependencies
poetry install

# Serve locally (http://localhost:8000)
poetry run mkdocs serve

# Build static site (always use --strict for link validation)
poetry run mkdocs build --strict
```
